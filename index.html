Easy

Secure, end-to-end encrypted chat from your terminal.

Download using npm:

npm install easy

Then type easy .

Secured and Encrypted.

  • ✓
    End-to-end encryption

    End-to-end encryption means messages are encrypted before they leave your device using a secret key. Even if someone is monitoring your network, or someone hacks our relay server, they would be unable to decipher your messages.

  • ✓
    Encrypted during transmission

    Easy uses encrypted websockets (TLS) to establish a secure connection between participants, and encrypts your messages once more during transmission.

  • ✓
    No message storage

    Easy messages are only routed in real-time to the recipient, and never stored or saved in a database. Once a message has been delivered, there's no record of that message ever being sent. Conversations on Easy mimick real life conversations: if you're not there when they said it, they'll have to repeat it once you get there.

What's Easy?

Easy makes it easy to create a 1-on-1 secure chat room using your Terminal.

Easy does away with the complexities of modern day secure chatting applications:

  • It has no remote message storage system, which means a message vanishes without a trace after it's delivered.
  • Encryption keys are set per-room, which means Easy doesn't have to deal with the usual complexity of issuing, managing, and securing keys. Instead, keys are communicated via a secure means of your choosing (in person, secure phone, password hint).
  • Easy uses high-entropy encryption "sentences" to encrypt messages. Instead of using random strings, you'll use easy-to-remember but impossible-to-guess passphrases like "the quick yellow dog jumped in the cold blue water".
  • Easy doesn't have a complicated UI that slows down messaging and communication.

Advanced Usage

To begin using Easy, just type easy after you've installed it. That should be all you need. The following custom options are available:

Launch Arguments

  • e or email: sign in with a different email than the one saved. Example: easy e=hello@standardnotes.org
  • logout: ends login session

Chat Commands When you're in a chat room, you can use these commands:

  • :set-secret: change the room encryption key
  • :show-secret: show the room encryption key
  • :server: print the address of the server you're connected to

Using Your Own Server

By default, Easy uses the secure server avaialble at https://api.easy.gd for both user authentication and websocket broadcoasting. To use your own server, set the following environment variables before launching Easy:

  • EASY_SERVER: Authentication server address.
  • EASY_SOCKET: Socket server address.
Both the authentication server and socket server are available as one Ruby application that you can clone from GitHub.
